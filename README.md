# warung-steak-bang-jago

Project website Warung Steak Bang Jago dengan 5 pilihan desain (Luxury, Modern, Rustic, Neo-Brutalism, Industrial).

## Pilihan Desain

- **Opsi 1**: Luxury Dark Steakhouse
- **Opsi 2**: Modern Elegant
- **Opsi 3**: Classic American Diner/Texas BBQ (Rustic Warm)
- **Opsi 4**: Playful Neo-Brutalism (Gaya Pop/Gen-Z)
- **Opsi 5**: Street Industrial (Warung Modern)

## Fitur

- Responsive & Mobile Friendly
- Design Switcher untuk berpindah antar opsi desain
- WA Order Integration
- Lucide Icons
